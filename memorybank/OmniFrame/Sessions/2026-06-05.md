# Session 2026-06-05

## Build Sheet prints to one 8.5x11 landscape page

Changed `KitBuildSheet.handlePrint` from `@page size: A4 landscape` to
`size: 11in 8.5in` (Letter landscape) and added a measure-and-scale
`fitToPage()` so the cover/build sheet always fits a single page even as TO
barcode + Authorized Ship Short rows grow. Wraps content in
`#print-scale-wrapper > #print-root`, lays it out at the printable width,
scales by `min(availW/w, availH/h, 1)`, and collapses the wrapper to the
scaled footprint so the transform doesn't leave a full-size box that
paginates. Single guarded `printOnce()`. `tsc` + ESLint clean; print-window
only. See [[Build-Sheet-Print-Fit-Letter-Landscape]].
