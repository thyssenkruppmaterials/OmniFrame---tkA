# Session — 2026-06-10

## Focus
User report: Location tab 3D map "loads all black, models don't show" + request
to make the scene robust and Minecraft-style buildable (staying isometric).

## Done
- **Diagnosed the black scene** with a new DEV-only repro route
  `/scene3d-harness` (synthetic layout, no auth; `?quality=` switch) + headless
  Playwright with Open-Meteo interception. Three compounding causes found:
  origin-anchored fog (scene ~60% fogged even in day), near-black night
  atmosphere (default weather location = Derby UK → night for US users), and
  drei `<SoftShadows>` PCSS intermittently corrupting all lit materials
  (~50% of loads). → [[Fix-Location-3D-Black-Scene-Night-Fog-SoftShadows]]
- **Fixes:** camera-relative fog via shared `isoCameraDistance()`; dusk-blue
  night grading + MIN_* light floors; editing forces neutral daylight;
  SoftShadows removed; CDN HDRI → procedural Lightformer `<Environment>` (no
  external fetch); `SceneCanvasBoundary` + WebGL context-lost auto-recovery.
- **Build mode shipped** → [[Build-Mode-Minecraft-Style-Placement]]: ghost
  preview, drag-to-paint, top/side face stacking, ⌥click delete, middle-click
  pick-block, R/Q/E rotate, wall/platform/stairs/ramp kinds.
